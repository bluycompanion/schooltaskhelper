# schooltaskhelper

Touch-first webbapp för att hjälpa barn (10–16 år) att planera och följa upp skoluppgifter.

## Fokus i första iterationen
- Visa alla aktuella uppgifter (ingen historikvy)
- Svårighetsgrad före planering
- Statusflöde med barn + förälderbekräftelse
- Gamification med hungerbar + avatar (emoji-placeholder)

## Struktur
- `apps/web` – frontend
- `docs` – beslut, miljöer, logg
- `packages` – gemensamma paket (vid behov)
